# [09] 3D bar → เชื่อม AI chat แบบ click-through

## สิ่งที่ต้องสร้าง

เชื่อม event กด 3D bar (ที่ stub ไว้ใน #03) กับ AI chat ฝั่งขวา เมื่อกดแท่ง ระบบแต่งคำถามภาษาไทยในรูปแบบ **"วิเคราะห์สต็อก {SKU} ที่ {สาขา} และแนะนำการจัดการ"** แล้วส่งเข้า chat เป็น user message โดยอัตโนมัติ ไม่ต้องพิมพ์เอง

## เกณฑ์การรับงาน

- [x] กดแท่งใดก็ได้ใน 3D View ส่งคำถามภาษาไทยที่แต่งขึ้นเข้า chat ทันที
- [x] คำถามระบุ SKU และชื่อสาขาที่ถูกต้องตามแท่งที่กด
- [x] AI ตอบพร้อม trust chip แสดง tool ที่ถูกเรียก
- [x] กดแท่ง RUN-A / Central Siam ได้ response ที่พูดถึง imbalance และปุ่ม action จาก #08 ปรากฏ
- [x] chat scroll ไปหาข้อความใหม่หลัง auto-submit

## ขึ้นกับ

- #03 — 3D View tab พร้อม stub click event ต้องมีก่อน
- #06 — chat ต้องรองรับการส่งข้อความแบบ programmatic
