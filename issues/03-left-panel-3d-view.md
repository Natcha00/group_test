# [03] ฝั่งซ้าย: tab 3D View

## สิ่งที่ต้องสร้าง

เพิ่ม tab **3D View** ในฝั่งซ้าย แสดง 3D bar chart ด้วย React Three Fiber + @react-three/drei ครอบคลุม 48 combinations (8 สาขา × 6 SKU) พร้อมกัน แกน X = สาขา, แกน Y (ความลึก) = SKU, ความสูง = stock on hand สีแท่งกราฟแสดงระดับความเร่งด่วน (คำนวณจาก on_hand ÷ avg_daily_sales):

- 🔴 แดง — < 7 วัน
- 🟡 เหลือง — 7–14 วัน
- 🟢 เขียว — ≥ 15 วัน

เมื่อกดแท่ง ให้ส่งคำถามภาษาไทยที่แต่งขึ้นเองเกี่ยวกับ SKU/สาขานั้นไปยัง AI chat ฝั่งขวา (ต้องรอ #06 ก่อนเชื่อมจริง — ถ้ายังไม่ merge ให้ log console และเชื่อมใน #09)
